About

Hasty is a SwiftUI-inspired static site generator for Dart.

  • Compose pages with hStack, vStack, zStack, and spacer
  • Type-safe styles via Style and Colors — no raw CSS strings
  • Full form, table, and list support
  • Built-in HTML escaping and prettified output
